# PIC32 Autonomous Racing Robot

Autonomous robot built on the PIC32 microcontroller for a timed race competition. The robot uses IR sensors for line following, PWM-controlled motors for movement, and a sound-triggered clap-start mechanism.

## What This Project Does

The robot listens for a clap to start, then follows a track autonomously using an array of IR sensors. Motor speed and direction are controlled via PWM outputs. The goal was to complete a timed race course as fast and accurately as possible.

## Technologies Used

- PIC32 microcontroller (MIPS architecture)
- IR reflectance sensors for line detection
- PWM motor control
- Clap-start sound detection logic
- C firmware (MPLAB X / XC32)
